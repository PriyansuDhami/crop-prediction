🌱 Crop Recommendation System Using Machine Learning is a web-based application that helps farmers select the most suitable crop based on soil nutrients (Nitrogen, Phosphorus, Potassium) and environmental conditions such as temperature, humidity, pH, and rainfall. The project evaluates multiple machine learning algorithms, including Logistic Regression, Support Vector Machine (SVM), LightGBM, and CatBoost, to compare their performance. Among them, CatBoost achieved the highest accuracy (98.86%), followed closely by LightGBM. The trained model is integrated into a Flask web application, where users enter soil and climate parameters to receive real-time crop recommendations. This system aims to improve crop productivity, reduce incorrect crop selection, and support data-driven, sustainable farming practices.
